# Biometric Face Attendance System

A modern Java-based biometric attendance management system developed using Java Swing. This project simulates a smart attendance workflow with a stylish graphical user interface, student registration, attendance tracking, and webcam integration workflow.

---

## 🚀 Features

* 🔐 Login System
* 👨‍🎓 Student Registration
* 📸 Biometric Face Scan Workflow
* 📝 Attendance Tracking
* 📂 File-Based Data Storage
* 🎨 Modern Java Swing User Interface
* 💻 VS Code Compatible
* ☁️ GitHub Hosted Project

---

## ⚙️ How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/ArunRam12-tech/Biometric-Face-Attendance-System.git
```

### 2. Open in VS Code

Open the project folder using Visual Studio Code.

### 3. Install Java Extension Pack

Install the following extension in VS Code:

* Extension Pack for Java

### 4. Run the Project

Open:

```text
src/Main.java
```

Click:

```text
Run
```

## 🧠 Project Workflow

1. User logs into the system
2. Student details are registered
3. Webcam biometric scan workflow starts
4. Attendance is marked automatically
5. Attendance records are stored in text files
