BIOMETRIC FACE ATTENDANCE SYSTEM

FEATURES:
- Stylish Java Swing UI
- Webcam biometric scan
- Face recognition module
- Attendance storage
- Student registration
- No MySQL required

IMPORTANT:
This project uses OpenCV for webcam access.

HOW TO RUN:
1. Install VS Code
2. Install Java Extension Pack
3. Download OpenCV Java library
4. Add OpenCV JAR to project
5. Open src/Main.java
6. Click Run

OpenCV Needed:
- opencv-xxx.jar
- opencv_javaxxx.dll
